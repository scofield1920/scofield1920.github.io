Unknown database“xxxx“idea连接数据库配置数据源时报错

发表于2022-10-28|更新于2022-10-11|MySQL
|阅读量:

问题产生：
今天用PHPstorm连接数据库时，配置数据源时，出现了这个问题

产生原因：
配置上述界面时，填写的数据库名称与你的电脑MySQL的名字不一致

​

​

问题解决：
1.命令行登陆MySQL

     D：\mysql-8.0.29-winx64\bin>mysql -u root -p      (前面是mysql的文件路径)

2.输入密码继续输入 show databases; （注意有 分号 ； ）

​

发现我安装的MySQL的名称是：mysql

  1. 所以再去修改idea里的数据库名称

    问题已解决，这个错误是数据库不存在或者是数据库名称写错了导致的，遇到这个错误不要害怕，冷静下来根据错误描述找问题。
